# Class Q Code of Conduct

Class Q is an open, collaborative, and global standard.  
We are committed to providing a welcoming and inclusive environment for all contributors.

---

## Our Standards

Participants are expected to:

- Show respect and professionalism  
- Provide constructive feedback  
- Collaborate in good faith  
- Avoid personal attacks or harassment  
- Keep discussions technical and focused  

---

## Unacceptable Behavior

- Harassment or discrimination  
- Disruptive or hostile behavior  
- Sharing proprietary or confidential information  
- Patent trolling or bad‑faith contributions  

---

## Enforcement

Violations may result in:
- warnings  
- temporary participation limits  
- removal from Working Groups  
- bans from the project  

Enforcement decisions are made by the Class Q Foundation governance team.

---

## Scope

This Code of Conduct applies to:
- GitHub discussions  
- Working Group meetings  
- Foundation events  
- All official Class Q communication channels
